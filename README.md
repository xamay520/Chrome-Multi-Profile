# Chrome 分身创建脚本

一键创建多个 Chrome 浏览器分身，每个分身拥有独立的 user-data-dir，数据互不干扰。

## 使用方法

1. 右键 Chrome分身创建脚本.ps1 → **使用 PowerShell 运行**
2. 桌面自动出现「浏览器分身」文件夹，内含快捷方式
3. 双击快捷方式即可打开独立 Chrome 分身

## 特性

- 自动查找 Chrome 安装路径（程序目录 + 注册表兜底）
- 优先将数据存储到 **D 盘**（D 不可用则 E，再不行才 C 盘用户目录）
- 每个分身独立 user-data-dir，可同时登录不同账号
- 分身数量可自定义：用记事本打开脚本，修改第一行 $count = 10 为想要的数字

## 效果

- 快捷方式：桌面\浏览器分身\分身 1.lnk ~ 分身 N.lnk
- 数据目录：D:\ChromeProfiles\Profile_1 ~ Profile_N（有 D 盘时）
